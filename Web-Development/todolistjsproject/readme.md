# Simple To-Do List Project

A basic web-based To-Do List application created using HTML, CSS, and JavaScript.

## Project Structure

The project consists of the following files:

- `index.html`: The HTML file that defines the structure of the To-Do List web page.
- `style.css`: The CSS file that styles the To-Do List.
- `script.js`: The JavaScript file that adds functionality to the To-Do List.

## Features

- Add tasks by entering text and clicking "Add Task" or pressing Enter.
- Remove tasks by clicking on them in the list.

## Usage

1. Open `index.html` in your web browser.

2. Enter a task in the input field and click "Add Task" or press Enter.

3. To remove a task, click on it in the list.
